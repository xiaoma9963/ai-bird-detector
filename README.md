# AI 智能驱鸟器 - 专利原型 App

## 📱 项目介绍

这是一个用于**专利原型演示**的 AI 智能驱鸟器手机应用。

### 核心功能
- 🦅 **AI 识别鸟类** - 使用深度学习模型实时识别鸟类
- 🚨 **声光报警** - 检测到鸟类后触发报警
- ⚡ **激光驱赶** - 模拟激光驱鸟效果
- 📊 **实时监控** - 显示识别结果和置信度

---

## 🚀 快速开始

### 方式一：下载 APK 安装（推荐）

1. 前往 [Actions](../../actions) 页面
2. 选择最新的构建任务
3. 在 Artifacts 中下载 `ai-bird-detector-apk`
4. 将 APK 传输到手机并安装

### 方式二：本地编译

#### 前置要求
- Python 3.10+
- Buildozer
- Android SDK/NDK

#### 编译步骤
```bash
# 1. 安装 Buildozer
pip install buildozer cython

# 2. 初始化（首次编译需要）
buildozer init

# 3. 编译 APK
buildozer android debug

# 4. APK 输出位置
# bin/ai-bird-detector-1.0.0-armeabi-v7a-debug.apk
```

## 🎯 使用说明

### 1. 启动应用
安装后点击图标启动，等待模型加载（约10秒）

### 2. 测试识别
- 将摄像头对准鸟类图片或实物
- 应用会自动识别并显示鸟类名称
- 检测到鸟类会触发报警和激光效果

### 3. 功能测试
- **测试报警**：点击底部"测试报警"按钮
- **测试激光**：点击底部"测试激光"按钮

## 🔧 技术架构

### AI 模型
- **模型**：MobileNetV3-Small（轻量级，适合手机）
- **训练数据**：ImageNet-1K（1000类物体）
- **鸟类识别**：覆盖常见鸟类约 100+ 种

### 识别流程
```
摄像头画面 → 图像预处理 → AI 模型推理 → 鸟类判断 → 报警/驱赶
```

### 核心参数
- 检测频率：每 5 帧（约 4 次/秒）
- 置信度阈值：50%
- 报警持续时间：3 秒

## 📋 专利相关

### 创新点
1. **AI + 驱鸟结合**：传统驱鸟器无智能识别
2. **精准识别**：只驱赶鸟类，不误伤其他动物
3. **声光复合驱赶**：报警 + 激光双重威慑
4. **实时监控**：可视化识别结果和统计数据

### 技术方案
```
摄像头采集 → AI 识别 → 鸟类判断 → 触发驱赶
    ↓           ↓          ↓          ↓
  实时画面   深度学习   置信度计算  声光报警
```

- 🏭 工厂、仓库

## ⚠️ 注意事项

1. **摄像头权限**：首次启动需要授权摄像头权限
2. **模型加载**：首次加载模型需要下载约 20MB
3. **识别准确率**：受光线、角度、距离影响
4. **激光安全**：当前为模拟效果，实际设备需注意激光功率
